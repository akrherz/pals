Memops: An Introduction to Memory Operations

In performing the exercises on the visible memory, you must use the load command. The command loads values into one location from another. The load command shown below would copy the value which is in location X[1] into location Z: 

  load z with x[1]
Note carefully the form of the command. Brackets [ ] and spaces are important. An l can be typed instead of the word load and a w can be typed instead of the word with.

Here are a few more examples of the load command: 

  load x[2] with x[5]
  load x[3] with z
Remember, you must press Return after typing a command to have it executed!
