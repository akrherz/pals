Welcome To Memops

An Introduction to Memory Operations

This lesson enables you to use some important computer concepts before they have been formally presented in class. By exploring concepts in this way, you should gain an intuitive understanding which will make formal learning easier and more meaningful.

The lesson has two parts. In both parts you will perform exercises in a model of the memory of a computer. In the first part, the values stored in memory will be visible. In the second part, the values stored in memory will be hidden from view.

Commands are entered using the editors at the bottom of each page. If letters you type do not appear in the editor, you need to click on the editor with the mouse.

When you have typed a complete command, you must press Return to have the command executed.

If you type an incorrect character, press Backspace to erase it.

When you have completed an exercise, click on the Check button on the bottom of the page and the computer will evaluate your solution. If your solution is correct, proceed to the next exercise. Otherwise, continue the current exercise and click on the Check button again when you think you have a correct solution.

You may need to reset the current exercise and start again with the original values in the array. To reset an exercise, click on the Reset button located at the bottom of the page.

If you get stuck at any time, return to the the preceding exercise. The steps required to solve the problems within one exercise are frequently part of the solution to the next.

The solutions will not be given to you. It is most important that you observe the model carefully and determine how it works.

The goal is to get started thinking about computers and how they work, not merely to solve these problems.

Do the visible exercises first!
